# 🦆 Friden AI Bot

Мощный Telegram-бот на базе **aiogram**, созданный для управления контентом и автоматизации. Проект запущен и работает через **GitHub Actions** 24/7.

## 🚀 Особенности
* **Безопасность:** Токен скрыт через GitHub Secrets и `.env`.
* **Автономность:** Бот работает в облаке, не требуя включенного компьютера или телефона.
* **Стек:** Python 3.10, aiogram, python-dotenv.
